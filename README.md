# 🏦 Loan Approval Predictor

A machine learning web application built with **Streamlit** that predicts whether a loan application will be approved or rejected based on applicant details.

## 🚀 Live Demo
> Run locally using the steps below

---

## 📌 Features

- Predicts loan approval using a trained Logistic Regression model
- Interactive UI with real-time prediction
- Shows approval/rejection confidence percentage
- Probability bar chart for visual understanding
- Input summary table after prediction

---

## 🧠 Model Info

| Detail | Info |
|---|---|
| Algorithm | Logistic Regression |
| Library | Scikit-learn |
| Features | 17 |
| Output | Approved (1) / Rejected (0) |

### Input Features
`Gender`, `Married`, `Dependents`, `Education`, `Self Employed`,
`Applicant Income`, `Co-Applicant Income`, `Loan Amount`,
`Loan Amount Term`, `Credit History`, `Property Area`
